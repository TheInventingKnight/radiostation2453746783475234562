How this station works

Streams of Grace runs from a private audio library in GitHub. A Cloudflare Worker reads a tracks.json file, builds a daily seeded playlist, and keeps every listener joined to the same hymn in a 24‑hour rotation.

• All MP3s stay private in your repo. • The browser only sees proxied stream URLs from the Worker. • The Worker caches each hymn in memory for smooth playback.
Tech stack: GitHub (private repo) for MP3s and tracks.json, a Cloudflare Worker for the synchronized playlist and streaming, and this static HTML player on GitHub Pages.
